## Sobre: 

Esse curso de SQL e Banco de Dados aborda :<br>
✔  **Entenda e escreva o T-SQL**<br>

✔  **Faça o tratamento de erro corretamente**<br>

Aulas do curso :<br>

✔  **Elementos do T-SQL**<br> 
✔  **Escrevendo funções**<br>
